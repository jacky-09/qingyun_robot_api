# qingyun_robot_api
这是我制作的基于`青云客api`和`requests`的**聊天机器人**封装产品。

## 快速使用
打开命令行，输入如下内容以安装依赖：
```bash
pip3 install requests
```
> 请读者确保已安装`Python3.8`，此处不在赘述。
接着，运行[`robot.py`](https://github.com/jackyfzh/qingyun_robot_api/blob/main/robot.py)以启动项目。

## 功能列表
- 聊天
- 坑爹 /doge
